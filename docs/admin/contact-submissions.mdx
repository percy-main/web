# Contact Submissions

<AdminTab name="Contact Submissions">
View messages submitted through the website's contact form.
</AdminTab>

## Overview

When visitors fill out a contact form on the website, their submissions appear here. This is a read-only view â€” there are no actions to take other than reading the messages.

## Searching

Use the search box to filter submissions by **name** or **email**.

## Submissions Table

| Column | Description |
|--------|-------------|
| **Name** | The person's name |
| **Email** | Their email address |
| **Message** | The message text (truncated to 100 characters) |
| **Page** | Which page the form was submitted from |
| **Date** | When the submission was made |

## Viewing Full Messages

Click any row to expand it and see the full message. Click again to collapse it. Messages preserve their original formatting (line breaks, etc.).

## Pagination

Submissions are shown 20 per page with Previous/Next navigation.
